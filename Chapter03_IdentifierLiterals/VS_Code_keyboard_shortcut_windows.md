# VS Code Keyboard Shortcuts (Windows)

| Action | Shortcut |
| --- | --- |
| New Window | `Ctrl+Shift+N` |
| Close Window | `Alt+F4` |
| Close Editor | `Ctrl+W` |
| Reopen Closed Editor | `Ctrl+Shift+T` |
| Split Editor | `Ctrl+\` |
| Focus Next Editor Group | `Ctrl+K Ctrl+Right` |
| Focus Previous Editor Group | `Ctrl+K Ctrl+Left` |
| Toggle Sidebar | `Ctrl+B` |
| Toggle Panel / Terminal | `Ctrl+` ` |
| Toggle Full Screen | `F11` |
| Toggle Zen Mode | `Ctrl+K Z` |
| Open Settings | `Ctrl+,` |
| Open Command Palette | `Ctrl+Shift+P` |
| Find | `Ctrl+F` |
| Replace | `Ctrl+H` |
| Save | `Ctrl+S` |
| Save All | `Ctrl+K S` |
