# VS Code Keyboard Shortcuts (macOS)

| Action | Shortcut |
| --- | --- |
| New Window | `Cmd+Shift+N` |
| Close Window | `Cmd+Q` |
| Close Editor | `Cmd+W` |
| Reopen Closed Editor | `Cmd+Shift+T` |
| Split Editor | `Cmd+\` |
| Focus Next Editor Group | `Cmd+K Cmd+Right` |
| Focus Previous Editor Group | `Cmd+K Cmd+Left` |
| Toggle Sidebar | `Cmd+B` |
| Toggle Panel / Terminal | `Ctrl+` ` |
| Toggle Full Screen | `Ctrl+Cmd+F` |
| Toggle Zen Mode | `Cmd+K Z` |
| Open Settings | `Cmd+,` |
| Open Command Palette | `Cmd+Shift+P` |
| Find | `Cmd+F` |
| Replace | `Cmd+Option+F` |
| Save | `Cmd+S` |
| Save All | `Cmd+Option+S` |
