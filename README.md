# LearnPlaywrightBatch2x

This repository contains JavaScript examples focused on basic programming concepts, identifier rules, and Visual Studio Code shortcut references.

## Structure

- `Chapter01_Basics/` - Basic JavaScript examples and Playwright setup verification.
- `Chapter02_JavascriptConcepts/` - JavaScript fundamentals and basic scripts.
- `Chapter03_IdentifierLiterals/` - Identifier naming rules, case styles, comments, and VS Code shortcut markdown files.
- `Chapter04_JavascriptConcepts/` - Advanced JavaScript concepts including variable declarations (var, let, const), function scoping, and hoisting.

## Notes

- `Chapter03_IdentifierLiterals/VS_Code_keyboard_shortcut_mac.md` contains macOS keyboard shortcuts.
- `Chapter03_IdentifierLiterals/VS_Code_keyboard_shortcut_windows.md` contains Windows keyboard shortcuts.

## Usage

Open the repository in Visual Studio Code to view and run the example scripts. The markdown files provide quick reference for VS Code keyboard shortcuts.
